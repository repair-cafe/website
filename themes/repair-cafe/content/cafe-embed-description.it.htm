Avete la possibilità di integrare questo caffé riparazione nella vostra pagina internet. Per far questo dovete inserire questo Snippet nella vostra pagina internet, nel posto in cui volete sia pubblicato.

<iframe src="{iframe_url_current_cafe}" id="repairCafe" width="100%" height="400" frameborder="0"></iframe>

Costruzione dell'URL

{iframe_url}

Scelta della lingua

La lingua può essere scelta con il parametro-URL :language. Le seguenti lingue sono a disposizione: {language_code_list}.

Scelta del caffé riparazione

Attraverso la modifica del parametro :cafe_slug URL si seleziona il caffé riparazione che deve essere mostrato. Il :cafe_slug può essere tolto dal URL di dettaglio die caffé.

Opzioni

Numero di eventi per pagina

Lo standard prevede {events_per_page_default} eventi per pagina. Per modifcarne il numero potete inserire un parametro events_per_page nell'URL.

Esempio:
{iframe_url_current_cafe}?events_per_page=5

iFrame Resizer (opzionale)

Nel iFrame si trova iFrame Resizer Caricate lo script nella versione 3.6.6. Così è possibile, che l'iFrame adegui automaticamente la grandezza al suo contenuto. In questo modo è possibile evitare inestetiche bande di scorrimento nella vostra pagina internet. Quello che dovete fare è riprendere nella vostra pagina lo script iframeResizer.min.js inizializzandolo per l’iFrame collegato:

<script>
iFrameResize({
    checkOrigin: ['{website_origin}']
}, '#repairCafe');
</script>

Con l'utilizzo degli iFrameResizers dovrebbe essere impostato anche l'attributo scrolling="no". In questo caso possono essere usati i seguenti esempi:

<iframe src="{iframe_url_current_cafe}" id="repairCafe" width="100%" height="400" frameborder="0" scrolling="no"></iframe>

Tutte le altre informazioni sui collegamenti con gli script iFrame Resizer si trovano nella documentazione ufficiale https://github.com/davidjbradshaw/iframe-resizer.
